# Alfido Tech Data Science Internship

## Overview

This repository contains the projects completed during my Data Science Internship at **Alfido Tech**. These projects demonstrate practical skills in data cleaning, exploratory data analysis, data visualization, and machine learning using Python and its data science ecosystem.

## Task 1 - Zomato Dataset Analysis

### Objective
Analyze restaurant data to identify factors influencing ratings, customer engagement, and restaurant popularity.

### Key Highlights
- Data Cleaning and Preprocessing
- Exploratory Data Analysis
- Correlation Analysis
- Data Visualization using Matplotlib and Seaborn
- Insights into restaurant ratings and customer preferences

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Task 2 - Loan Approval Prediction

### Objective
Build a machine learning model to predict loan approval based on borrower information.

### Key Highlights
- Data Preprocessing and Feature Engineering
- Label Encoding and Feature Scaling
- Train-Test Split
- Logistic Regression
- Decision Tree Classification
- Model Evaluation using ROC-AUC Score

### Model Performance

| Model | ROC-AUC Score |
|---------|-------------|
| Logistic Regression | 0.833 |
| Decision Tree | 0.852 |

### Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
