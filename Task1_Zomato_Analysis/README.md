# Zomato Dataset Analysis

## Objective
Analyze restaurant data to identify factors influencing ratings, customer engagement, and restaurant popularity.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Data Cleaning
- Checked missing values
- Removed duplicate records
- Converted ratings into numeric format

## Exploratory Data Analysis
- Restaurant category distribution
- Online order analysis
- Table booking analysis
- Rating distribution
- Cost vs Rating analysis
- Correlation heatmap

## Key Findings
- Dining is the most common restaurant category.
- Online ordering restaurants generally receive better ratings.
- Table booking facilities are associated with higher ratings.
- Popular restaurants attract significantly more votes.
- Restaurant cost has only a weak relationship with ratings.
