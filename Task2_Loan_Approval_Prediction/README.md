# Loan Approval Prediction

## Objective
Build a machine learning model to predict loan approval using borrower information.

## Dataset
Loan Approval Prediction Dataset

## Steps Performed
- Data Loading
- Data Cleaning
- Label Encoding
- Feature Scaling
- Train-Test Split
- Logistic Regression
- Decision Tree
- Model Evaluation

## Results

| Model | ROC-AUC |
|---------|---------|
| Logistic Regression | 0.833 |
| Decision Tree | 0.852 |

## Conclusion
Decision Tree achieved the highest ROC-AUC score and was selected as the best model.
